Code4rena

Smart Contract Auditing Platform

Code4rena is a smart contract auditing platform. Inventing the competitive security audit, crypto projects get wider coverage than traditional security reviews, leveraging a distributed network of security researchers.

We built an entirely new platform to facilitate competitive audits. Security researchers audit code and submit vulnerabilities, and through a system of incentives, a neutral judge reviews them for quality and validity. Participants are paid for unique, valid findings, which are shared with customers to be mitigated before deploying their code to production.
